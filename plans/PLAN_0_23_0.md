## Milestone Goals

- Close the gap between differential refresh O(Δ) theory and observed scaling
- Fix three failure modes: threshold collapse, early collapse, structural bug
- Add diagnostic GUCs for delta SQL inspection and tuning
- Improve planner statistics for change buffer tables
- Document DVM query complexity limits
